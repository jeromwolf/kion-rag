## 2. 리얼 KION 사이트 vs PoC 비교

### 2.1 기능 비교

| 기능 | KION 리얼 사이트 | PoC 시스템 |
|-----|-----------------|-----------|
| **검색 방식** | 키워드 검색 + 테이블 목록 | 자연어 AI 챗봇 |
| **검색 결과** | 장비명, 제조사, 모델, 담당자 | 장비명, 스펙, 추천 이유, 점수 |
| **필터링** | 기관별 선택 | AI 자동 필터 (웨이퍼/온도/재료) |
| **응답 형태** | 정적 테이블 | 실시간 스트리밍 설명 |
| **추천 기능** | 없음 | AI 기반 맞춤 추천 |
| **예약 연동** | 별도 페이지 이동 | 카드에서 바로 예약 페이지 |
| **장비 수** | 124+ (한국나노기술원만) | 100개 (샘플 데이터) |

### 2.2 장점 비교

#### KION 리얼 사이트
- 실제 장비 데이터 보유
- 담당자 연락처 직접 제공
- 공식 예약 시스템 연동
- 모든 소속 기관 장비 통합

#### PoC 시스템 (우리 시스템)
- **자연어 검색**: "6인치 Si 열처리 장비" 같은 자연어로 검색
- **AI 추천**: 사용자 요구사항 분석 후 최적 장비 추천
- **스펙 기반 필터링**: 웨이퍼 크기, 온도, 재료 자동 파싱
- **추천 이유 설명**: 왜 이 장비가 적합한지 상세 설명
- **실시간 스트리밍**: 응답을 실시간으로 확인
- **매칭 점수**: 적합도를 퍼센트로 표시

### 2.3 개선 필요 사항

| 항목 | 현재 상태 | 개선 방향 |
|-----|---------|----------|
| 장비 데이터 | 샘플 100개 | 실제 KION 데이터 연동 |
| 중국어 필터링 | 97% 성공 | 100% 필터링 목표 |
| 응답 시간 | 평균 5.16초 | 3초 이하 목표 |
| 추천 정확도 | 단일 추천 다수 | 복수 추천 강화 |
| 예약 연동 | 데모 페이지 | 실제 KION 연동 |
